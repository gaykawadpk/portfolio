# PK. — Portfolio ✨🔍

> *"When you have eliminated the impossible, whatever remains, however improbable, must be the truth."*

A **Sherlock Holmes × Taylor Swift** themed personal portfolio for **Puja Kumari** — Senior Software Engineer, Photographer & Poet.

## 🌐 Live Site

[**gaykawadpk.github.io/portfolio**](https://gaykawadpk.github.io/portfolio/)

## 🔍 Features

- **Cinematic page loader** with magnifying glass animation
- **Interactive constellation background** that reacts to mouse
- **Custom cursor** with gold ring + dot
- **3D tilt effect** on skill & poem cards
- **Film strip gallery** with Ken Burns zoom, "EVIDENCE" tags, and cinematic lightbox
- **Code-cracking title decode** animation on hover
- **Magnetic buttons** that subtly follow your cursor
- **Fingerprint scanner** animation on profile photo
- **Scroll progress bar** + **back-to-top** button
- **Keyboard shortcuts** (1-5 for sections, D for theme toggle)
- **Konami Code easter egg** (↑↑↓↓←→←→BA)
- **Console easter egg** — open DevTools for a surprise
- **Dark / Light theme** toggle with persistent preference
- **Mobile responsive** with touch swipe in lightbox
- **Morse code badge** (.--. -.-) in the navbar

## 🛠️ Tech Stack

- Pure **HTML5**, **CSS3**, **JavaScript** — no frameworks, no dependencies
- Google Fonts (Inter, Playfair Display, JetBrains Mono)
- Hosted on **GitHub Pages**
